## Quiénes somos

Append es una empresa de desarrollo especializada en construir soluciones digitales completas: backend, frontend, infraestructura, diseño de experiencias y librerías reutilizables.  
Nuestros productos y plataformas combinan ingeniería sólida con una identidad visual consistente.

Trabajamos bajo principios de claridad, orden, diseño simple y una arquitectura orientada a módulos.

---

## Tecnología y Enfoque

**Backend:**  
.NET 8, Web API, IdentityServer, EF Core, SQL Server, PostgreSQL.

**Frontend:**  
Angular 19–20, NX Monorepos, Microfrontends, TypeScript, Storybook.

**Infraestructura:**  
Docker, Ubuntu Server, Nginx, GitHub Actions, CI/CD, AWS, servidores dedicados.

---

## Ecosistema Append

Estamos construyendo un conjunto de librerías y módulos reutilizables:

**Librerías internas (NPM / Monorepo):**
- `@append/ui` – componentes reutilizables y estándar visual unificado  
- `@append/theme` – tokens, colores, tipografía, espaciado  
- `@append/core` – servicios base, manejo de estado, auth  
- `@append/user` – módulo genérico de usuarios, permisos y administración  

Este ecosistema permite desarrollar nuevas aplicaciones más rápido, manteniendo consistencia visual y técnica.

## Proyectos en desarrollo

### Turnos Médicos Multiempresa  
Plataforma de gestión de turnos, profesionales, pacientes, sesiones, agendas y multiclínicas. Construida sobre arquitectura modular para escalabilidad.

### Foundar  
Plataforma de crowdfunding educativo y tecnológico.  
Estética espacial moderna, experiencias inmersivas y herramientas de comunidad.

### VPcam  
Sistema de captura deportiva basado en cámaras IP, routers dedicados, sincronización remota y procesamiento de videos.

### AppStore Empresarial  
Marketplace de módulos plug-and-play para acelerar soluciones empresariales.
